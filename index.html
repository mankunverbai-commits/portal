Courses


HTML
HTML (HyperText Markup Language) is the main structure of a webpage. It is used to create headings, paragraphs, images, links, and other parts of a website. It works like the skeleton of a body that gives shape and structure to the content.


CSS
CSS (Cascading Style Sheets) is used to style and design the webpage. It adds colors, fonts, spacing, and layout to make the page look attractive. It works like clothes and decoration for the HTML structure.


Javascript
JavaScript is a programming language that makes a webpage interactive and alive. It helps things move or change when users click, type, or scroll. It works like the brain that adds logic and action to the webpage.
